# PHP Social Media App

## Description
This is a social media web application built with PHP, MySQL, HTML, and CSS. The app focuses on backend functionality including user authentication, profile management, posts, inbox messaging, and user search.

## Features
- User registration and login with password hashing
- Create posts
- Profile management and profile picture upload
- Inbox messaging between users
- Search and view other users
- Responsive and simple UI (focus on backend logic)

## Technologies Used
- PHP
- MySQL
- HTML / CSS

## How to Run
1. Clone the repository
2. Import the MySQL database file into your local server (XAMPP/WAMP)
3. Update `Database.php` with your database credentials
4. Run the project on a local PHP server
5. Access via browser (e.g., `http://localhost/php-social-media-app/`)

## Notes
- Focus was primarily on backend functionality; UI is basic
- Profile pictures and posts are stored in the database and/or local `uploads` folder
- Feel free to modify UI styling
